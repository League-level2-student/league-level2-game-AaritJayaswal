Pong

by: Aarit Jayaswal

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because although it's very simple, it is a fun game to play.

2. What I Learned

I received feedback on making a way to win the game so now I added code where if the player gets 10 points they win.

3. Architecture of My Game

Using the ball class, I set up the parameters of the ball and how it should move. In the paddle and OpponentPaddle classes, I made code to make the paddles move. The GamePanel class is where I add all of the objects to the screen and where the game checks if the player is pressing any keys. Finally, the Pong class is where the panel is added to the window and I can run the game.

4. Technical Implementation

I am especially proud of making the ball bounce when it hits any of the paddles. For this, I created 3 collision boxes, one for the player's paddle, one for the ball, and one for the opponent's paddle. When the ball touches either paddle, it reverses its direction and heads towards the opposite side of the screen. Also, when the ball collides with the top or bottom edge of the screen, it bounces.

5. Most Challenging Part

The most challenging part of this project was figuring out how to make the game end if the ball crosses the player's paddle. This took a long time because I wasn't sure how to make the game react if the ball crosses the edge of the screen. I overcame this by adding a few lines of code where if the ball's x value is equal to the x value of the edge of the screen, the state of the game is switched to the end.

6. Future Fixes and Updates

In the future, I would like to make the game have different levels where the ball moves faster. I would also like to make the movement of the paddles smoother.
